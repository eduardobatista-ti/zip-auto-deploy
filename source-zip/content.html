Bem-vindo ao setup inicial do AutoDeploy

Faça o deploy do seu projeto no GitHub no seu servidor! A cada Git Push, o servidor atualiza automaticamente. Aproveite esse projeto desenvolvido por Eduardo Batista e mantido pela ZipCloud

Execução do deploy:

O deploy é feito em Shell e executado via PHP. Por isso é crucial que o PHP instalado esteja com o shell_exec habilidado no php.ini, mas se estiver desabilitado, é muito simples habilitar.

Sim! Para que o deploy seja executado com sucesso, o git precisa estar instalado no servidor. Mas não se preocupe! Servidores Linux costumam vir com o git instalado, e é bem simples de instalar.

Essa interface gráfica vai deixar de existir. Logo após o seu primeiro deploy, todos os componentes visuais serão removidos, apenas os diretórios /auto-deploy, /deploy-logs e o aquivo deploy-config.php vão ser mantidos junto com seu porojeto!
